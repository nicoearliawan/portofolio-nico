## [1.0.0] - 2026-05-12

### Added
- Initial release
- Hero section with profile and status badges
- Overview section with contact information
- Social media links
- About section with bio
- GitHub contributions calendar
- Tech stack showcase with icons
- Experience timeline (work & education)
- Projects showcase with filtering
- Blog section with latest posts
- Footer with scroll-to-top button
- Dark/Light mode toggle with localStorage
- Command Palette (Ctrl+K / Cmd+K)
- Responsive design (mobile-first)
- Smooth animations with Framer Motion
- React Router for navigation
- Blog page
- Resume page
- Store page
- Comprehensive documentation
